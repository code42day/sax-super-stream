# sax-super-stream

Transform stream implemented using SAX with hierarchical parsing

## Install

```sh
$ npm install --save sax-super-stream
```

## Usage

```js
var saxSuperStream = require('sax-super-stream');

saxSuperStream('Rainbow');
```
